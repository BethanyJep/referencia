# 🎨 Sketchnote Creation Guide

Sketchnotes are visual summaries that combine drawings, text, and diagrams to explain complex concepts in a simple, memorable way.

## What Makes a Great AI Sketchnote?

### Essential Elements:
1. **Title** - Clear lesson name at the top
2. **Key Concepts** - Main ideas in boxes or bubbles
3. **Visual Metaphors** - Icons and drawings that represent concepts
4. **Flow** - Arrows showing relationships between ideas
5. **Color** - Strategic use to highlight important points
6. **White Space** - Don't overcrowd, leave breathing room

## Tools You Can Use

### Digital Tools (Recommended):
- **Excalidraw** (Free, web-based) - https://excalidraw.com
- **Figma** (Free tier available) - Great for clean designs
- **Procreate** (iPad, paid) - Best for hand-drawn feel
- **Concepts** (iPad/Desktop) - Infinite canvas
- **Adobe Illustrator** - Professional option

### Hand-Drawn:
- Draw on paper with markers/pens
- Scan or photograph with good lighting
- Use a photo editing app to enhance contrast

## Recommended Dimensions

- **Width**: 1200-1600px
- **Height**: 900-1200px (or longer for scrollable content)
- **Format**: PNG (for transparency) or JPG
- **Resolution**: 72-150 DPI for web

## Tips for Data Science Sketchnotes

### For AI Concepts:
- Use brain icons for learning/intelligence
- Gears for processing
- Networks/graphs for neural networks
- Flow charts for algorithms
- Boxes and arrows for data flow

### For ML Concepts:
- Draw simple datasets as dots/clusters
- Show decision boundaries
- Use trees for decision trees
- Layers for neural networks
- Input → Process → Output diagrams

### For LLMs:
- Stacked boxes for transformer layers
- Attention as spotlight/rays
- Tokens as puzzle pieces
- Context window as a frame
- Training as a journey/path

## Example Workflow

1. **Brainstorm** (5-10 min)
   - List all key concepts from the lesson
   - Think of visual metaphors for each

2. **Sketch Draft** (10-15 min)
   - Rough layout on paper or digitally
   - Arrange elements logically
   - Add arrows to show flow

3. **Refine** (15-30 min)
   - Clean up lines
   - Add color strategically
   - Ensure text is legible
   - Balance visual elements

4. **Export & Test** (5 min)
   - Export at correct dimensions
   - View on different devices
   - Ensure readability at various sizes

## Templates

Create a personal template with:
- Your consistent header style
- Standard icon set for common concepts
- Color scheme
- Typography choices

This makes creating new sketchnotes faster!

## Quick Start

Don't have time for elaborate drawings? Start simple:

### Minimal Sketchnote:
1. Clean title at top
2. 3-5 main concepts in boxes
3. Simple icons (circles, squares, arrows)
4. One accent color
5. Clear text

Even a simple visual summary is better than text alone!

## Accessibility Note

Always provide:
- Alt text describing the sketchnote content
- Text caption summarizing key points
- Ensure sufficient color contrast

---

**Remember**: Sketchnotes don't have to be perfect art. They're learning tools! Focus on clarity and helpfulness over artistic perfection. 🎨
