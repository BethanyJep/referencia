📖 Introduction: What are Diffusion Models?

Imagine a beautiful painting that slowly gets covered in TV static noise until it's completely random. Now imagine a cat with a magic eraser that can carefully remove that noise, step by step, until the original painting emerges!

That's the essence of diffusion models! They learn to reverse the process of adding noise. The model trains on images that have been gradually corrupted with noise, learning at each step how to "denoise" the image. At generation time, it starts with pure noise and gradually cleans it into a beautiful, coherent image!

💡 Key Insight

Diffusion Models work by learning to reverse the process of adding noise. Training: gradually add noise to images. Generation: start with noise and gradually remove it to create images!

🐱 The Magic Eraser Story

Training time: The cat watches as beautiful paintings are slowly covered in static. At each step, it memorizes: "When the noise looks like THIS, the image underneath probably looks like THAT."

Generation time: Given pure static, the cat thinks: "Hmm, this noise pattern suggests there might be a face here... let me erase a bit... yes, I see an eye forming!" Step by step, it reveals an image that was never really there—but COULD have been!

With text prompts: "A cat astronaut on the moon" guides the erasing: "This noise should become a helmet here... moon surface there..."

📚 Core Content

📉 Forward Process (Fixed)

What: Gradually add Gaussian noise over T steps

When: During training only

Formula: q(x_t | x_{t-1}) - add a bit more noise

Result: Original image → Pure noise

This process is deterministic—we don't learn it!

📈 Reverse Process (Learned)

What: Learn to remove noise step by step

When: Training (learn) & Generation (use)

Formula: p_θ(x_{t-1} | x_t) - predict & remove noise

Result: Pure noise → Generated image

This is what the neural network learns!

# Simplified diffusion concept
# Forward: Add noise
def forward_step(image, noise_level):
    noise = torch.randn_like(image)
    return sqrt(1 - noise_level) * image + sqrt(noise_level) * noise

# Reverse: Predict and remove noise (THIS is learned!)
def reverse_step(noisy_image, model, timestep):
    predicted_noise = model(noisy_image, timestep)
    # Subtract the predicted noise to get cleaner image
    return denoise(noisy_image, predicted_noise)

Training Objective: Predict the Noise!

  1. Take a clean image from the training set
  2. Pick a random timestep t (1 to T, e.g., 1000)
  3. Add noise appropriate for that timestep
  4. Ask the model: "What noise was added?"
  5. Compare predicted noise vs. actual noise
  6. Update weights to predict better

The Denoising Loop

  1. Start with pure random noise (x_T)
  2. For t = T down to 1:
    • Model predicts what noise is in current image
    • Subtract (some of) that predicted noise
    • Add a tiny bit of new noise (for stochasticity)
  3. After T steps: Beautiful generated image!
# Simplified generation (sampling)
def generate_image(model, num_steps=1000):
    # Start with pure noise
    x = torch.randn(1, 3, 256, 256)  # Random noise image
    
    # Gradually denoise
    for t in reversed(range(num_steps)):
        # Model predicts the noise
        predicted_noise = model(x, t)
        
        # Remove some of that noise
        x = remove_noise(x, predicted_noise, t)
        
        # Add tiny bit of randomness (except last step)
        if t > 0:
            x = x + small_noise()
    
    return x  # Clean generated image!

Adding Text Conditioning

Text-to-image models add a crucial ingredient: text embeddings that guide the denoising!

  1. Encode the text prompt (using CLIP or similar)
  2. Feed text embedding to the model at each step
  3. Model learns: "Given this noise AND this text, what should I denoise toward?"

Why Diffusion Won (For Now)

Diffusion models have largely overtaken GANs for image generation because:

  • Training is much more stable—no balancing act needed
  • They produce more diverse outputs
  • Quality matches or exceeds GANs
  • Easy to add conditioning (text, images, etc.)

The main downside: slower generation (though techniques like DDIM and distillation are helping!)

💻 Run Stable Diffusion Locally

# Using Hugging Face Diffusers library
from diffusers import StableDiffusionPipeline
import torch

# Load the model (downloads ~4GB first time)
pipe = StableDiffusionPipeline.from_pretrained(
    "runwayml/stable-diffusion-v1-5",
    torch_dtype=torch.float16
)
pipe = pipe.to("cuda")  # Use GPU

# Generate an image!
prompt = "A majestic cat wearing a tiny crown, oil painting style"
image = pipe(prompt).images[0]

# Save it
image.save("royal_cat.png")
print("Your royal cat has been generated! 👑🐱")
