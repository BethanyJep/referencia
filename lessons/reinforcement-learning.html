📖 Introduction: What is Reinforcement Learning?

Imagine a cat navigating a maze 🐱🧀. The cat doesn't have a map—it has to explore! When it finds cheese, it gets a treat (+1 reward). When it hits a dead end or gets sprayed with water, that's a penalty (-1). Over time, the cat learns which paths lead to treats!

That's Reinforcement Learning (RL)—learning through trial and error. An agent takes actions in an environment, receives rewards or penalties, and learns a policy to maximize total reward over time!

💡 Key Insight

Reinforcement Learning trains agents to make sequences of decisions by rewarding good outcomes and penalizing bad ones. The agent learns a strategy (policy) to maximize cumulative rewards!

🐱 The Maze Cat Story

Day 1: Cat enters maze, randomly wanders. Dead end → spray bottle! 💦 Cat remembers: "Don't go there."

Day 10: Cat has explored more. Found cheese twice going left! 🧀 Cat thinks: "Left seems good..."

Key insight: No one TOLD the cat where to go. It learned through experience—trial, error, and rewards!

📚 Core Content

🐱 Agent

The learner/decision-maker. Our maze cat!

🌍 Environment

The world the agent interacts with. The maze!

📍 State (s)

Current situation. "I'm at position (2,3)"

🎬 Action (a)

What the agent can do. Up, down, left, right

🎁 Reward (r)

Feedback signal. +1 for cheese, -1 for spray

📋 Policy (π)

The learned strategy. "At state s, do action a"

The RL Loop

while not done:
    state = observe_environment()      # Where am I?
    action = policy.choose(state)      # What should I do?
    new_state, reward = env.step(action)  # Do it, see result
    policy.learn(state, action, reward, new_state)  # Update!
    state = new_state

🔍 Exploration

"Try something new! There might be better cheese!"

Taking random or uncertain actions to discover new information.

Cat tries a new path it's never taken

💰 Exploitation

"Stick with what works! I know this path has cheese!"

Using current knowledge to maximize reward.

Cat takes the path it knows leads to cheese

ε-Greedy Strategy

With probability ε: explore (random action)

With probability 1-ε: exploit (best known action)

epsilon = 0.1  # 10% exploration
if random() < epsilon:
    action = random_action()    # Explore!
else:
    action = best_known_action()  # Exploit!

Often ε starts high (explore more early) and decreases over time (exploit learned knowledge).

Q-Learning (Value-Based)

Learn the value of each state-action pair: Q(s,a)

"How good is it to take action a in state s?"

# Q-Learning update rule
Q[state][action] += learning_rate * (
    reward + gamma * max(Q[new_state]) - Q[state][action]
)
# Translation: Update Q-value based on actual reward + future value

The agent builds a Q-table: a lookup for "in this state, this action is worth X."

Policy Gradient (Policy-Based)

Directly learn the policy π(a|s) without Q-values.

"Learn a function that outputs action probabilities."

Good for continuous action spaces (e.g., robotics).

Actor-Critic (Hybrid)

Actor: Learns the policy (what to do)

Critic: Learns value function (how good is this?)

Critic helps train the Actor more efficiently!

Popular versions: A2C, A3C, PPO, SAC

Deep RL

Use neural networks to approximate Q-functions or policies.

DQN: Deep Q-Network (neural net as Q-table)

PPO: Proximal Policy Optimization (stable policy gradient)

Enables RL in complex environments like video games!

🤖 LLM Alignment (RLHF)

Making ChatGPT helpful! Reward model trained on human preferences.

Reinforcement Learning from Human Feedback

RLHF uses RL to align LLMs with human preferences!

  1. Pre-train LLM on text (next-token prediction)
  2. Fine-tune on demonstrations (supervised learning on good examples)
  3. Train a reward model:
    • Show humans two responses to same prompt
    • "Which response is better?"
    • Train a model to predict human preferences
  4. RL training:
    • LLM generates response (action)
    • Reward model scores it (reward)
    • Update LLM to generate higher-scored responses

💻 Train Your Own Agent

# Simple Q-Learning for a cat maze!
import numpy as np

# Initialize Q-table (states × actions)
Q = np.zeros((num_states, num_actions))
epsilon = 0.1
alpha = 0.1  # learning rate
gamma = 0.99  # discount factor

for episode in range(1000):
    state = env.reset()  # Cat enters maze
    
    while not done:
        # ε-greedy action selection
        if np.random.random() < epsilon:
            action = np.random.randint(num_actions)  # Explore
        else:
            action = np.argmax(Q[state])  # Exploit
        
        # Take action, observe result
        new_state, reward, done = env.step(action)
        
        # Q-learning update
        Q[state, action] += alpha * (
            reward + gamma * np.max(Q[new_state]) - Q[state, action]
        )
        
        state = new_state

print("Cat learned the maze! 🐱🧀")
