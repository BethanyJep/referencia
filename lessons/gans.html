📖 Introduction: What are GANs?

Imagine two cats in an eternal creative battle. One cat is an artist (🎨 Generator) trying to paint perfect forgeries of famous artworks. The other is a critic with a detective hat (🔍 Discriminator) trying to spot the fakes from the real masterpieces.

At first, the artist is terrible—the critic easily spots every fake. But they keep competing, and both get better! Eventually, the artist creates such convincing work that even the expert critic can't tell real from fake. That's a GAN!

💡 Key Insight

GANs (Generative Adversarial Networks) pit two neural networks against each other: a Generator that creates fake data, and a Discriminator that tries to distinguish fake from real. Through this competition, both improve!

"GAN: Artist vs. Critic, may the best cat win!"

📚 Core Content

🎨 Generator (G)

Job: Create fake data that looks real

Input: Random noise (latent vector)

Output: Fake image/data

Goal: Fool the Discriminator

Loss: Gets punished when D correctly identifies fakes

# Generator creates from noise
noise = torch.randn(batch_size, 100)
fake_images = generator(noise)

🔍 Discriminator (D)

Job: Distinguish real from fake

Input: Real or fake data

Output: Probability (0=fake, 1=real)

Goal: Correctly classify all inputs

Loss: Gets punished for wrong classifications

# Discriminator judges
real_score = discriminator(real_images)
fake_score = discriminator(fake_images)

Training Loop:

  1. Train Discriminator:
    • Show it real images → Should output 1 (real)
    • Show it fake images from G → Should output 0 (fake)
    • Update D to be better at telling them apart
  2. Train Generator:
    • Generate fake images
    • Pass through D (but don't update D!)
    • G wants D to output 1 for its fakes
    • Update G to create more convincing fakes
  3. Repeat thousands of times!
# Simplified GAN training loop
for epoch in range(num_epochs):
    for real_images in dataloader:
        # === Train Discriminator ===
        # Real images should be classified as 1
        real_loss = criterion(D(real_images), ones)
        
        # Fake images should be classified as 0
        noise = torch.randn(batch_size, latent_dim)
        fake_images = G(noise)
        fake_loss = criterion(D(fake_images.detach()), zeros)
        
        d_loss = real_loss + fake_loss
        d_optimizer.step()  # Update discriminator
        
        # === Train Generator ===
        # Generator wants D to think fakes are real (output 1)
        noise = torch.randn(batch_size, latent_dim)
        fake_images = G(noise)
        g_loss = criterion(D(fake_images), ones)  # Wants 1s!
        
        g_optimizer.step()  # Update generator

DCGAN

Deep Convolutional GAN - Uses CNNs for stable image generation

Key: First architecture that worked well for images

StyleGAN

Generates ultra-realistic faces with style control

Key: Can control features at different levels (pose, identity, fine details)

CycleGAN

Translates between domains without paired examples

Key: Horse↔Zebra, Photo↔Painting, Summer↔Winter

Pix2Pix

Image-to-image translation with paired data

Key: Sketch→Photo, Day→Night, Edges→Real images

ProGAN

Progressive Growing - Starts small, adds layers

Key: More stable training for high-resolution images

BigGAN

Class-conditional, massive scale

Key: Generate specific ImageNet classes with incredible detail

Mode Collapse 😱

Problem: Generator only produces a few types of outputs (finds one trick that fools D)

Solution: Minibatch discrimination, unrolled GANs, feature matching

Training Instability 📉

Problem: G and D don't improve together; one dominates

Solution: Careful learning rates, Wasserstein loss, spectral normalization

Vanishing Gradients 👻

Problem: If D is too good, G gets no useful signal

Solution: Label smoothing, WGAN gradient penalty

Evaluation Difficulty 🤷

Problem: Hard to measure how good generated images are

Solution: FID score, Inception Score, human evaluation

💻 Build a Simple GAN

# Simple GAN architecture in PyTorch
import torch.nn as nn

class Generator(nn.Module):
    """The Artist Cat 🎨"""
    def __init__(self, latent_dim=100):
        super().__init__()
        self.model = nn.Sequential(
            nn.Linear(latent_dim, 256),
            nn.LeakyReLU(0.2),
            nn.Linear(256, 512),
            nn.LeakyReLU(0.2),
            nn.Linear(512, 784),  # 28x28 image
            nn.Tanh()  # Output between -1 and 1
        )
    
    def forward(self, z):
        return self.model(z)

class Discriminator(nn.Module):
    """The Critic Cat 🔍"""
    def __init__(self):
        super().__init__()
        self.model = nn.Sequential(
            nn.Linear(784, 512),
            nn.LeakyReLU(0.2),
            nn.Linear(512, 256),
            nn.LeakyReLU(0.2),
            nn.Linear(256, 1),
            nn.Sigmoid()  # Output probability: 0=fake, 1=real
        )
    
    def forward(self, img):
        return self.model(img)
