📖 Introduction: What are Neural Networks?

Imagine a classroom where cats pass notes to each other. The first row of cats receives a message (input), processes it, and passes their notes to the cats behind them. Each cat adds its own interpretation before passing it on. By the time the note reaches the back of the room, the message has been transformed into an answer!

That's essentially how neural networks work! They're computing systems inspired by the brain, made up of interconnected nodes (neurons) that process and pass information through layers.

💡 Key Insight

Neural Networks are layers of interconnected nodes that transform input data step-by-step. Each connection has a weight that determines how much influence one node has on another.

"NN: Cats passing notes, each adding their perspective!"

  • How neurons work (the basic building block)
  • The layer structure: input, hidden, and output layers
  • How information flows forward through the network

🐱 The Cat Network Story

Input cats: The first row receives raw data—pixels from an image. Each cat looks at different parts: one sees edges, one sees colors, one sees shapes.

Hidden cats: They combine notes from the input cats. "I got 'pointy ears' + 'whiskers' = probably a face!" Each passes their combined insights forward.

Output cat: Receives all the processed notes and makes the final call: "CAT!" or "NOT CAT!"

Learning: When wrong, a messenger cat runs back through the network saying "We need to adjust!" — that's backpropagation! 🏃‍♂️📝

📚 Core Content

A single neuron is the basic building block. Think of it as one cat receiving notes:

How a Neuron Works:

  1. Receive inputs: Gets values from previous neurons (or raw data)
  2. Apply weights: Each input is multiplied by a weight (importance factor)
  3. Sum it up: Add all weighted inputs together, plus a bias
  4. Activation function: Decide whether to "fire" (pass the message on)
# A single neuron calculation
import numpy as np

def neuron(inputs, weights, bias):
    # Step 1-2: Weight the inputs
    weighted_sum = np.dot(inputs, weights) + bias
    
    # Step 3: Activation (ReLU example)
    output = max(0, weighted_sum)  # If positive, pass it; else, 0
    return output

# Example: 3 inputs coming from previous neurons
inputs = [0.5, 0.3, 0.8]
weights = [0.4, 0.6, 0.2]  # Learned importance
bias = 0.1

result = neuron(inputs, weights, bias)
# This cat decides what to pass to the next layer!

📥 Input Layer

Receives raw data. For an image: one neuron per pixel. For text: one neuron per word/token.

The cats that first see the data

🧠 Hidden Layer(s)

Where the magic happens! These neurons learn patterns and features. More layers = more complex patterns learned.

Cats discussing and combining notes

📤 Output Layer

Produces the final answer. For classification: one neuron per category. For regression: single value.

The cat that announces the answer

"Deep" Learning

When a network has many hidden layers, we call it a deep neural network. That's where "Deep Learning" comes from!

More layers can learn more abstract concepts:

  • Layer 1: Edges and simple shapes
  • Layer 2: Parts (eyes, ears, noses)
  • Layer 3: Faces
  • Layer 4: Specific individuals

Activation functions decide if a neuron should "fire" (be excited 😺) or stay quiet (sleepy 😴).

ReLU (Rectified Linear Unit)

f(x) = max(0, x)

If positive → pass it through. If negative → output 0.

Cat: "Only pass good vibes!" 😺

Sigmoid

f(x) = 1 / (1 + e^-x)

Squashes output between 0 and 1. Good for probabilities!

Cat: "How likely on a scale of 0-1?" 🤔

Softmax

Converts outputs to probabilities that sum to 1.

Used in output layer for multi-class classification.

Cats voting: "30% cat, 65% dog, 5% rabbit" 🗳️

Tanh

f(x) = (e^x - e^-x) / (e^x + e^-x)

Squashes between -1 and 1. Centered at zero.

Cat: "Strong no to strong yes" 😾↔️😺

➡️ Forward Pass

Data flows from input to output. Each neuron computes its value and passes it forward. At the end, we get a prediction.

Cats passing notes front to back 📝→📝→📝

📊 Calculate Loss

Compare prediction to the correct answer. The "loss function" measures how wrong we were.

Loss = How bad was our prediction?

  • MSE: Mean Squared Error (for regression)
  • Cross-Entropy: For classification

⬅️ Backward Pass (Backpropagation)

The "messenger cat" runs backward through the network, telling each cat how to adjust their weights to make fewer mistakes next time.

Key insight: Uses calculus (chain rule) to figure out how much each weight contributed to the error.

Messenger cat: "You weighted that input too heavily! Turn it down!" 🏃‍♂️📝

🔧 Gradient Descent

The algorithm that actually updates the weights. Takes small steps in the direction that reduces the loss.

Learning Rate: How big each step is (too big = overshoot, too small = slow learning)

Dense / Fully Connected (MLP)

Every neuron connects to every neuron in the next layer.

Use: Tabular data, basic classification

Convolutional (CNN)

Specialized for images. Uses filters that slide across the image to detect features.

Use: Image classification, object detection

Recurrent (RNN/LSTM)

Has memory! Processes sequences by feeding output back as input.

Use: Time series, early NLP

Transformer

Uses attention mechanism. Powers modern LLMs like GPT!

Use: Text generation, translation, LLMs

💻 Build Your First Neural Network

# Simple neural network with PyTorch
import torch
import torch.nn as nn

class CatClassifier(nn.Module):
    def __init__(self):
        super().__init__()
        # Input layer → Hidden layer 1 (cats receiving images)
        self.layer1 = nn.Linear(784, 128)  # 28x28 image = 784 pixels
        # Hidden layer 1 → Hidden layer 2 (cats processing)
        self.layer2 = nn.Linear(128, 64)
        # Hidden layer 2 → Output (cat announcing result)
        self.layer3 = nn.Linear(64, 2)  # Cat or Not Cat
        
        self.relu = nn.ReLU()  # Activation: only positive vibes!
    
    def forward(self, x):
        # Forward pass: notes flowing through the network
        x = self.relu(self.layer1(x))  # First cats process
        x = self.relu(self.layer2(x))  # Second group processes  
        x = self.layer3(x)             # Final cat decides
        return x

# Create the network
model = CatClassifier()
print(f"Total parameters (note weights): {sum(p.numel() for p in model.parameters())}")
