📖 Introduction: What are AI Agents?

Imagine Agent Whiskers 🕵️🐱 — a cat with a tiny spy earpiece, night-vision goggles, and a utility belt. You give it a mission: "Book me a flight to Paris and find a cat-friendly hotel." Agent Whiskers doesn't just answer—it takes action! It searches flights, compares prices, books the best option, and emails you the confirmation!

AI Agents are systems that can perceive their environment, make decisions, and take autonomous actions to achieve goals. Unlike chatbots that just respond, agents can use tools, plan multi-step tasks, and execute complex workflows!

💡 Key Insight

AI Agents combine LLMs with the ability to take actions: calling APIs, searching the web, executing code, and using tools. They can reason, plan, and act autonomously!

"Agent: LLM + Tools + Autonomy = Can actually DO things!"

📚 Core Content

💬 Chatbot (LLM)

Input: Text prompt

Output: Text response

Can: Answer questions, generate text

Cannot: Take real actions, access live data

Example: "What's the weather?" → "I don't have access to current weather data."

🕵️ Agent (LLM + Tools)

Input: Goal/Task

Output: Actions + Results

Can: Use tools, execute plans, take actions!

Has: Access to APIs, code execution, memory

Example: "What's the weather?" → *calls weather API* → "It's 72°F and sunny!"

🧠 Brain (LLM)

The reasoning engine. Understands goals, plans steps, decides which tools to use.

🛠️ Tools

APIs and functions the agent can call: search, code execution, databases, etc.

💾 Memory

Short-term (conversation) and long-term (vector DB) to remember context.

📋 Planning

Breaking complex tasks into steps, creating and revising plans.

# Agent with tools (conceptual)
agent = Agent(
    llm=GPT4(),
    tools=[
        WebSearch(),
        Calculator(),
        EmailSender(),
        CodeExecutor(),
        CalendarAPI()
    ],
    memory=VectorMemory(),
    planner=ReActPlanner()
)

# Give it a task!
result = agent.run("Find the best pizza place nearby and book a table for tonight")

How Tool Calling Works

  1. Define tools: Describe available functions to the LLM
  2. LLM decides: Based on the task, which tool to call?
  3. Execute: System runs the function with LLM's parameters
  4. Return result: Feed output back to LLM
  5. Continue: LLM decides next step based on result
# OpenAI Function Calling example
tools = [{
    "type": "function",
    "function": {
        "name": "get_weather",
        "description": "Get current weather for a location",
        "parameters": {
            "type": "object",
            "properties": {
                "location": {"type": "string", "description": "City name"}
            }
        }
    }
}]

response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "What's the weather in Paris?"}],
    tools=tools
)

# LLM responds with: call get_weather(location="Paris")
# You execute it, return result, LLM formulates final answer!

ReAct (Reason + Act)

Interleave reasoning and action:

Thought: I need to find flight prices
Action: search_flights("NYC to Paris")
Observation: Found 5 flights, cheapest is $450
Thought: That's good! Now I need hotels...
Action: search_hotels("Paris, pet-friendly")
...

Plan-and-Execute

Create a full plan first, then execute steps:

  1. Plan: [Search flights, Book flight, Search hotels, Book hotel, Send confirmation]
  2. Execute step 1...
  3. Execute step 2...
  4. (Replan if needed)

Multi-Agent Systems

Multiple specialized agents working together:

  • Researcher Agent: Gathers information
  • Writer Agent: Creates content
  • Critic Agent: Reviews and improves
  • Coordinator: Manages the team

Like a team of cat agents, each with a specialty! 🐱🐱🐱

💻 Build a Simple Agent

# Simple ReAct Agent with LangChain
from langchain.agents import create_react_agent, AgentExecutor
from langchain_openai import ChatOpenAI
from langchain.tools import DuckDuckGoSearchRun, Tool

# Create tools for Agent Whiskers
search = DuckDuckGoSearchRun()
tools = [
    Tool(
        name="Search",
        func=search.run,
        description="Search the web for current information"
    )
]

# Create the agent
llm = ChatOpenAI(model="gpt-4")
agent = create_react_agent(llm, tools, prompt)
agent_executor = AgentExecutor(agent=agent, tools=tools, verbose=True)

# Give Agent Whiskers a mission!
result = agent_executor.invoke({
    "input": "What are the top 3 cat cafes in Tokyo and their ratings?"
})

print(result)  # Agent searches, compiles info, and reports back! 🕵️🐱
